Goals of these units

  • Create a web-page that asks for a name and outputs it.
  • Using the if..else construct, write the code which asks: ‘What is the “official” name of JavaScript?’ If the visitor enters “ECMAScript”, then output “Right!”, otherwise – output: “Didn’t know? ECMAScript!
  • Using if..else, write the code which gets a number via prompt and then shows in alert:
    • 1, if the value is greater than zero,
    • -1, if less than zero,
    • 0, if equals zero.
  • Write the code which asks for a login with prompt. If the visitor enters "Admin", then prompt for a password, if the input is an empty line or Esc – show “Canceled.”, if it’s another string – then show “I don’t know you”. The password is checked as follows:
    • If it equals “TheMaster”, then show “Welcome!”,
    • Another string – show “Wrong password”,
    • For an empty string or cancelled input, show “Canceled.”
  • Rewrite 'if..else' into '?'
